# Personal Website Template (GitHub Pages)
  
A minimal, professional, -themed personal website starter.  
  
## Files
- `index.html` – homepage
- `projects.html` – projects page
- `resume.html` – resume page (embeds PDF)
- `style.css` – styling/theme
- `assets/` – images + resume PDF

## Quick Start (Workshop)
1. Click **Use this template** (or fork this repo).
2. Rename your repo to `yourusername.github.io` (for personal sites)  
   - OR keep the name for a project site.
3. Go to **Settings → Pages**
   - Source: Deploy from branch
   - Branch: `main`
   - Folder: `/root`
4. Your site will be live at:
   - Personal site: `https://yourusername.github.io`
   - Project site: `https://ORG.github.io/REPO/`

## Customize
- Update name + links in `index.html`
- Add projects in `projects.html` (copy/paste a project card)
- Replace:
  - `assets/profile.jpg`
  - `assets/resume.pdf`
- Change colors in `style.css` under `:root`

## Notes
- Keep descriptions short.
- Add 2–4 strong projects, not 12 weak ones .
